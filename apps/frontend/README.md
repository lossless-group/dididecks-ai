# DidiDecks Frontend

This is the tracked SvelteKit frontend package for DidiDecks inside the repo workspace.

## Package root

- `apps/frontend`

## Commands

- `pnpm --filter @dididecks/frontend dev`
- `pnpm --filter @dididecks/frontend build`
- `pnpm --filter @dididecks/frontend check`

## Notes

- Shared contracts come from `packages/shared`.
- Static deploy assets live under `static/`.
- Product routes and components live under `src/`.
